## 🎓 HSK 1 Curriculum

### Overview

**HSK 1 Requirements:**
- 150 words
- 45 grammar points
- Basic daily conversations
- Reading simple sentences
- Writing simple characters

**Target Students:**
- Complete beginners
- No prior Chinese knowledge
- Want to pass HSK 1 exam
- Learn practical conversation

---

### 10-Lesson Structure

**Lesson 1: Greetings & Introductions (你好)**
```
Vocabulary (15 words):
• 你 (nǐ) - you
• 好 (hǎo) - good
• 你好 (nǐ hǎo) - hello
• 我 (wǒ) - I, me
• 是 (shì) - to be
• 中国 (Zhōngguó) - China
• 人 (rén) - person
• 学生 (xuésheng) - student
• 老师 (lǎoshī) - teacher
• 名字 (míngzi) - name
• 叫 (jiào) - to call/be called
• 什么 (shénme) - what
• 吗 (ma) - question particle
• 呢 (ne) - particle
• 谢谢 (xièxie) - thank you

Grammar:
• Basic sentences with 是 (shì)
• Questions with 吗 (ma)
• Questions with 什么 (shénme)

Dialogues:
A: 你好!
B: 你好!
A: 你叫什么名字?
B: 我叫李明。你呢?
A: 我叫王芳。

Practice:
• Introduce yourself
• Ask someone's name
• Say thank you
```

**Lesson 2: Numbers (数字)**
```
Vocabulary (15 words):

• 几 (jǐ) - how many
• 个 (gè) - measure word
• 多少 (duōshao) - how much/many
• 块 (kuài) - yuan (money)

Grammar:
• Counting with measure words
• Questions with 几 and 多少

Dialogues:
A: 这个多少钱?
B: 五块钱。
A: 我要三个。

Practice:
• Count to 100
• Ask prices
• Say phone numbers
```

## 🎬 Video Production

### Equipment Needed

**Minimum (Budget: $0-50):**
- ✅ Smartphone (iPhone/Android)
- ✅ Good lighting (natural window light)
- ✅ Quiet room
- ✅ Free software (CapCut, DaVinci Resolve)

**Recommended (Budget: $100-300):**
- ✅ Webcam (Logitech C920)
- ✅ USB microphone (Blue Yeti or Fifine K669)
- ✅ Ring light ($30-50)
- ✅ Green screen (optional)
- ✅ Video editing software (CapCut, Premiere Pro)

---

### Recording Setup

**Location:**
```
• Quiet room (no background noise)
• Clean background (wall or bookshelf)
• Good lighting (soft, from front)
• Comfortable chair
• Tripod or stable surface for camera
```

**Camera Settings:**
```
• Resolution: 1080p minimum (4K if possible)
• Frame rate: 30fps
• Orientation: Horizontal (landscape)
• Focus: Manual (stay focused on you)
```

**Audio Settings:**
```
• Use external mic if possible
• Record in quiet environment
• Test audio levels before recording
• Aim for clear, natural voice
• No background music during speaking
```

---

### Video Structure (Per Lesson)

**Intro (30 seconds):**
```
• Greeting: "Xin chào! Chào mừng đến với HoaNgữ"
• Lesson title: "Hôm nay chúng ta học Bài 1: Chào hỏi"
• Learning objectives: "Bạn sẽ học được..."
• Enthusiasm & energy!
```

**Vocabulary Section (5-7 minutes):**
```
• Show word on screen (Chinese character)
• Show pinyin
• Pronounce clearly (2-3 times)
• Show Vietnamese meaning
• Give example sentence
• Encourage repetition
```

**Example:**
```
老师: "Từ đầu tiên: 你好"
Screen: 你好 (nǐ hǎo)
老师: "nǐ hǎo... nǐ hǎo" (say slowly)
Screen: Nghĩa: Xin chào
老师: "你好 nghĩa là xin chào. Hãy lặp lại với tôi: nǐ hǎo"
Pause for repetition
老师: "Tuyệt vời! Tiếp theo..."
```

**Grammar Section (3-5 minutes):**
```
• Explain grammar point simply
• Show structure/pattern
• Give 3-5 examples
• Highlight common mistakes
• Practice sentences
```

**Dialogue Section (3-4 minutes):**
```
• Present full dialogue on screen
• Read through once (natural speed)
• Break down line by line
• Explain context
• Practice with pauses for repetition
• Role-play demo
```

**Practice Section (2-3 minutes):**
```
• Interactive exercises
• Pause for students to answer
• Give immediate feedback
• Encourage active participation
```

**Closing (1 minute):**
```
• Recap key points (3-5 items)
• Homework assignment
• Next lesson preview
• Motivational message
• Call to action: "Hãy luyện tập và hẹn gặp lại ở bài sau!"
```

**Total video length: 15-20 minutes**

---

### Recording Tips

**Delivery:**
- ✅ Speak clearly and slowly
- ✅ Smile and show energy
- ✅ Make eye contact with camera
- ✅ Use hand gestures naturally
- ✅ Pause between sections
- ✅ Encourage and motivate

**Technical:**
- ✅ Record in short segments (5-10 min each)
- ✅ Can re-record if mistake
- ✅ Leave pauses for editing
- ✅ Check audio/video before long recording
- ✅ Save frequently

**Content:**
- ✅ Stick to lesson plan
- ✅ Use simple Vietnamese
- ✅ Repeat key points 2-3 times
- ✅ Give plenty of examples
- ✅ Keep students engaged

---

### Editing Process

**Software:** CapCut (free) or DaVinci Resolve (free)

**Steps:**

1. **Import footage**
   - Import all video clips
   - Import audio (if separate)
   - Organize in timeline

2. **Cut & arrange**
   - Remove mistakes
   - Cut long pauses
   - Arrange in order
   - Add B-roll (if any)

3. **Add on-screen text**
   - Chinese characters (large, clear font)
   - Pinyin (smaller, under characters)
   - Vietnamese meaning (bottom)
   - Use consistent styling

4. **Add subtitles**
   - Full Vietnamese subtitles
   - Chinese subtitles for dialogues
   - Sync accurately
   - Use readable font (sans-serif, white with black outline)

5. **Add graphics**
   - Lesson title screen
   - Section headers
   - Vocabulary cards
   - Grammar diagrams
   - Practice questions

6. **Add background music**
   - Soft, non-distracting
   - Lower volume during speaking
   - Upbeat during intro/outro
   - Use royalty-free music (YouTube Audio Library)

7. **Color correction**
   - Adjust brightness
   - Fix white balance
   - Consistent look

8. **Export**
   - Resolution: 1080p
   - Format: MP4
   - Codec: H.264
   - Quality: High
   - File size: Keep under 500MB

## ✍️ Content Writing

### Lesson Plan Template

**For each lesson, write:**

```markdown
# Lesson [X]: [Topic Name]

## Learning Objectives
By the end of this lesson, students will be able to:
- [ ] Objective 1
- [ ] Objective 2
- [ ] Objective 3

## Vocabulary (15 words)
1. Word (pinyin) - meaning - example sentence
2. ...

## Grammar Points (3-5 points)
1. Grammar rule
   - Explanation
   - Structure
   - Examples (3)
   - Common mistakes

## Dialogues (2-3)
**Dialogue 1: [Situation]**
A: ...
B: ...
A: ...

[Explanation]
[Vocabulary notes]

## Practice Exercises
1. Fill in the blank (10 questions)
2. Translation (10 sentences)
3. Listening comprehension (5 questions)
4. Speaking practice (3 topics)

## Homework
- Review vocabulary (flashcards)
- Practice dialogues (record yourself)
- Complete workbook exercises
- Watch video 2 times

## Additional Resources
- Printable vocabulary list (PDF)
- Audio files (MP3)
- Flashcard deck (Anki)
- Quiz (online)
```

### Quiz Creation

**Per lesson: 20 questions**

**Question Types:**

1. **Multiple choice (10 questions)**
```
Q: "你好" nghĩa là gì?
A. Tạm biệt
B. Xin chào ✓
C. Cảm ơn
D. Xin lỗi
```

2. **Fill in the blank (5 questions)**
```
Q: 我___王明。(Tôi tên là Vương Minh)
A: 叫
```

3. **Translation (3 questions)**
```
Q: Translate to Chinese: "Bạn tên là gì?"
A: 你叫什么名字？
```

4. **Listening (2 questions)**
```
Q: Listen and choose correct pinyin
Audio: [你好]
A. nǐ hǎo ✓
B. ní hǎo
C. nǐ háo
D. ní háo
```

## 🎓 Tips for Success

**Content Creation:**
- Start with what you know best
- Don't aim for perfection on first course
- Get feedback from native speakers
- Test with real students
- Iterate based on data

**Time Management:**
- Batch similar tasks
- Record multiple lessons in one session
- Edit in dedicated blocks
- Don't get stuck on small details
- Ship > Perfect

**Staying Motivated:**
- Remember your why
- Celebrate small wins
- Take breaks when needed
- Ask for help
- Enjoy the process!
